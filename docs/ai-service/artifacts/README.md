# Artifact Layout

This folder is initialized in Phase 0 so later phases can deposit evidence in a predictable structure.

- `datasets/`: raw, cleaned, synthetic, and sequence datasets
- `models/`: model weights, vocab files, training configs
- `plots/`: generated PNG charts
- `tables/`: CSV tables and metrics
- `eval/`: evaluation metrics and benchmark outputs
- `reports/`: phase reports and artifact manifests
- `screenshots/`: UI and graph screenshots
- `benchmarks/`: load, latency, and throughput outputs
